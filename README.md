# Causal and Physics-aware Data-Driven Framework for Optimizing Gear Heat Treatment processes.

Data-driven and physics-aware modeling of steel heat treatment, with a focus on quenching, phase transformation, and causal constraints.

## Goal
This repository rebuilds and extends a prior thesis workflow into a clean, reproducible research codebase.

## Initial focus
- Reproduce baseline thesis results
- Clean the data and modeling pipeline
- Introduce causal awareness
- Introduce physics-aware constraints
- Start from isolated thermal processes, especially quenching

## Structure
- `src/` reusable Python code
- `notebooks/` exploratory notebooks
- `data/` raw and processed data
- `configs/` experiment settings
- `tests/` validation tests
- `docs/` project notes and methodology
