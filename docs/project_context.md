# Project context

## Origin
This repository rebuilds and extends a prior thesis on data-driven modeling of heat treatment for AISI 9310 steel.

## Original thesis focus
- Predict distortion
- Predict hardness
- Predict phase fractions
- Use FEM-generated data
- Compare ML/DL models against simulated and experimental results

## Upgrade direction
- Introduce causal awareness
- Introduce physics-aware constraints
- Start from isolated thermal processes
- Focus first on quenching
- Explore carbon-dependent KM
- Explore latent heat / energy consistency
